# Hermes target

```
python -m tools.build hermes
```

Output: `build/hermes/ncplot-hermes/`.

Near-identity copy of `src/skills/` plus a Hermes MCP config snippet. The
SKILL.md format is already compatible (agentskills.io), so no translation
is needed.

See the generated README inside the build artifact for install steps.
